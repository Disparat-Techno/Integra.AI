Perguntas Frequentes (FAQ)

O que é Integra.AI?

Integra.AI é uma ferramenta CLI e Web que utiliza inteligência artificial (Google Gemini) para automatizar a geração de código para integrações com APIs. Ela pode criar clientes de API, exemplos de uso e testes.

Como instalar o Integra.AI?

A maneira mais fácil de instalar o Integra.AI no Windows é usando o script de instalação PowerShell fornecido. Este script irá construir o pacote (se necessário) e instalá-lo automaticamente.

1. Abra o PowerShell no diretório raiz do projeto (c:\Integra.AI).

2. Execute o script de instalação:

.\scripts\install.ps1

Alternativamente, se você já tem o arquivo .whl (disponível nas Releases do GitHub) e o pipx instalado, você pode instalar diretamente:

pipx install integra_ai-0.1.0-py3-none-any.whl
Como configurar a chave da API Gemini?

Para usar a funcionalidade de IA, você precisa definir sua chave da API Google Gemini como uma variável de ambiente. Substitua <SUA_CHAVE> pela sua chave real:

Windows (PowerShell):

$env:GEMINI_API_KEY="<SUA_CHAVE>"

Linux/macOS (Bash):

export GEMINI_API_KEY="<SUA_CHAVE>"

Recomenda-se adicionar isso ao seu arquivo de perfil (ex: .bashrc, .zshrc, .profile) para que seja carregado automaticamente.

Como gerar código usando a CLI?

Use o comando integra ai com um prompt descrevendo a integração desejada:

integra ai --prompt "Gerar integração com a API Uber Direct para criar entregas"

Você também pode especificar o modelo Gemini a ser usado:

integra ai --prompt "Gerar integração com a API Uber Direct" --model gemini-pro
Como usar o Painel Web?

Para iniciar o painel web, execute o seguinte comando no diretório raiz do projeto:

python -m flask --app web.app run --debug

Em seguida, acesse http://127.0.0.1:5000/ (ou a porta configurada) no seu navegador. Lá você encontrará uma interface para interagir com a IA.

Quais modelos Gemini são suportados?

Atualmente, o Integra.AI suporta gemini-1.5-flash (padrão) e gemini-pro. Você pode especificar o modelo usando o argumento --model na CLI ou selecionando-o no painel web.

Como executar os testes unitários?

Para executar os testes unitários do projeto, navegue até o diretório raiz do projeto (c:\Integra.AI) no seu terminal e execute:

python -m unittest discover tests

Isso descobrirá e executará todos os testes na pasta tests.
