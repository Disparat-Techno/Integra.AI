Integra.AI

Automatize integrações com APIs usando IA (Gemini)

Ver no GitHub Baixar (Windows)

CLI + Painel Web

Gere clientes de API, exemplos de uso e testes com um comando. Analise documentação (URL/Texto) e deixe a IA fazer o resto.

Gere com IA

Use a API Gemini para criar código pronto para consumo de endpoints.

integra ai --prompt "Gerar integração com a API Uber Direct"

Cliente rápido

Crie um cliente Python simples para testar sua API.

integra connect --name meu-cliente --base-url https://api.exemplo.com

Teste imediato

Dispare requisições e valide respostas rapidamente.

integra test --name meu-cliente --endpoint /status

Recursos

  • Autocompletar e corrigir códigos de integração
  • Geração automática de clientes, exemplos e testes
  • Histórico de integrações e armazenamento local
  • Templates prontos (Python) e suporte a Node
  • Logs e auditoria de operações

Como usar

  1. Instale: pipx install integra_ai-0.1.0-py3-none-any.whl (veja Releases)
  2. Defina a chave: $env:GEMINI_API_KEY="<SUA_CHAVE>"
  3. Gere código: integra ai --prompt "Gerar integração com X"
  4. Abra o painel: python -m flask --app web.app run --debug
